# Use Case #

Я как пользователь хочу видеть меню, чтобы заказать пиццу.

## * ** Основной сценарий ** * ##

1. Пользователь заходит на сайт.

2. На главной странице пользователь заходит в меню "Пиццы".

3. Система перенаправляет пользователя на страницу с ассортиментом всех пицц.

4. Пользователь кликает на карточку товара.

5. В карточке товара находится изображение пиццы и краткое описание.

6. Пользователь закрывает карточку товара.

7. Пользователь, использую быстрые кнопки добавляет соответствующие пиццы в корзину.

8. В открывшемся модальном окне пользователь выбирает размер пиццы, толщину теста, топпинг и добавляет свой выбор в корзину.

9. Пользователь заходит в корзину видит свой заказ и добавленные опции к пицце, видит сумму заказа.

10. Пользователь видит ранее введенный адрес доставки.

11. Пользователь выбирает способ оплаты из списка.

12. Пользователь выбирает время доставки и подтверждает заказ нажав на кнопку "Оформить заказ".

13. Система отображает Экран со списком пицц.
 
 * Экран отображает номер заказа.
  
 * Экран отображает дату заказа.

 * Экран отображает время доставки.

 * Экран отображает тип оплаты.

 * Экран отображает состояние заказа.

 * Экран отображает общую сумму заказа.

 * Экран отображает опции к заказу.

14. Система передает заказ Администратору.

15. Администратор перенаправляет заказ на кухню, система отображает данное действие в статусе.

16. Курьер забирает готовый заказ, система отображает данное действие в статусе.

17. Курьер передает заказ пользователю и отмечает в системе как "Заказ доставлен".
